## How to Use

Each concept has its own markdown file in `notes/` explaining the 
syntax, rules and examples. Practice queries are in `practice/` 
and can be run on [sqliteonline.com](https://sqliteonline.com).

## Key Concepts Summary

| Concept | What it does |
|---------|-------------|
| SELECT | Retrieve data from a table |
| WHERE | Filter rows by condition |
| ORDER BY | Sort results |
| GROUP BY | Group rows and apply aggregates |
| JOIN | Combine data from multiple tables |
| Subquery | Query inside another query |
| INSERT | Add new rows |
| UPDATE | Modify existing rows |
| DELETE | Remove rows |
| CREATE TABLE | Define a new table structure |
