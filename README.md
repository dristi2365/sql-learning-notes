# SQL Learning Notes 📊

A structured collection of SQL notes, concepts and practice queries 
documenting my learning journey from basics to real-world usage.

## Structure

```
sql-learning-notes/
    notes/          # Concept notes in markdown
    practice/
        queries.sql # All practice queries with comments
    README.md
```

## Topics Covered

- [ ] SELECT Basics
- [ ] WHERE Filtering
- [ ] ORDER BY and LIMIT
- [ ] GROUP BY and Aggregate Functions
- [ ] JOINS
- [ ] Subqueries
- [ ] Real World Practice

## How to Use

Each concept has its own markdown file in `notes/` explaining the 
syntax, rules and examples. All practice queries are collected in 
`practice/queries.sql` and can be run on 
[sqliteonline.com](https://sqliteonline.com).
